# System Info Viewer

A Python script that displays basic system information.

## Description
This project retrieves and displays details about the system using Python's built-in platform module.

## Features
- OS detection
- Platform details
- Processor information
- Python version display

## Technologies Used
- Python
- platform module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python system_info.py

## Example Output

OS: Windows  
Platform: Windows-10-10.0.19045  
Processor: Intel64 Family 6 Model 142  
Python Version: 3.11.0
